BDS Toolbox: Data Visualization

Instructor – Behavioral Data Science Master's, University of Amsterdam, 2024-2025

This module is part of the BDS Toolbox, a course within the Behavioral Data Science Master's programme at the University of Amsterdam. The module covers effective data visualization using R and ggplot2, interactive visualizations, and version control with Git and GitHub.

Course website  ·  Slides  ·  GitHub repository

Topics covered:

  • Principles of effective data visualization
  • Static visualizations with ggplot2
  • Interactive visualizations with plotly
  • Collaborative workflows with Git and GitHub

Programming: The Next Step

Instructor – Research Master's in Psychology, University of Amsterdam, 2024

A four-week intensive course in which students independently develop a software package from start to finish in either R or Python. The course emphasises good software — not just functional software — with a focus on package structure, documentation, vignettes, and unit testing. Each week students submit a pull request on GitHub and give peer feedback on their classmates' work, culminating in a live demonstration of the finished package.

Topics covered:

  • R and Python package development
  • Writing documentation and vignettes
  • Unit testing
  • Version control and code review with Git and GitHub
